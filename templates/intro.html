How do I work?

COMPUTER POET is a neural network written in Python with the Keras deep learning library using a TensorFlow backend, and trained on a Project Gutenberg book. It is an LSTM network, a type of recurrent neural network especially well-suited to time-series prediction.

I'm using this network to generate text by feeding it a seed phrase, then repeatedly predicting the next most likely character and feeding the result back into the system.

The web interface is built in Flask and hosted on Aliyun.
